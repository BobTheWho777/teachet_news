# Новостной сайт для техникума "Бизнес и Право"

## Описание
Приложение для публикации новостей для преподавателей техникума "Бизнес и Право".

## Возможности
- Просмотр новостей на главной странице
- Категории важности новостей (низкая, средняя, высокая)
- Админ-панель для добавления новостей
- Загрузка картинок для новостей
- Авторизация администратора

## Требования
- PHP 7.4+
- MySQL 5.7+
- Веб-сервер (Apache/Nginx)

## Установка

### 1. База данных
Импортируйте SQL файл:
```bash
mysql -u root -p < database.sql
```

Или выполните SQL запросы вручную через phpMyAdmin.

### 2. Настройка подключения
Откройте `config/database.php` и измените параметры подключения к БД:
```php
$host = 'localhost';
$dbname = 'teachers_news';
$username = 'root';
$password = 'ваш_пароль';
```

### 3. Права доступа
Создайте папку для загрузки изображений и дайте права на запись:
```bash
mkdir uploads
chmod 777 uploads
```

### 4. Запуск
Поместите файлы в директорию веб-сервера и откройте в браузере:
```
http://localhost/teachers_news/
```

## Структура проекта
```
├── config/
│   ├── database.php      # Настройки подключения к БД
│   └── constants.php     # Общие константы проекта
├── includes/
│   ├── header.php        # Универсальный header
│   └── footer.php        # Универсальный footer
├── admin/
│   ├── login.php         # Страница входа
│   ├── auth.php          # Обработка авторизации
│   ├── dashboard.php     # Админ-панель (CRUD)
│   └── logout.php        # Выход из системы
├── uploads/              # Папка для загруженных изображений
├── index.php             # Главная страница с пагинацией и фильтрами
├── news.php              # Страница отдельной новости
├── database.sql          # SQL схема
└── README.md
```

## Технологии
- PHP (backend)
- MySQL (база данных)
- Tailwind CSS (стилизация)
